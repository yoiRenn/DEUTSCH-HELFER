csv具体格式要求（可以给ai的prompt）
```
请扮演一个专业的 OCR 和数据格式化专家。我需要将德语教材的单词表图片转换为特定的 CSV 格式，以便导入我的背单词程序。

请识别图片中的单词、词性、中文释义、复数/变位形式以及例句（如果有），并严格按照以下规则输出 CSV 代码块。

### 1. CSV 表头（第一行必须是这个）：
type,gender,word,cn,forms,example

### 2. 列的填写规则：
1. **type (词性)**: 
   - 统一转换为标准英文缩写：
   - 名词 -> n
   - 动词 -> v
   - 形容词 -> adj
   - 副词 -> adv
   - 介词 -> prep
   - 连词 -> conj
   - 代词 -> pron
2. **gender (词性-仅名词填写)**:
   - 如果是名词 (n)，根据书上的标识 (der/die/das 或 m/f/n) 填写完整的冠词：`der`, `die` 或 `das`。
   - 如果不是名词，此项**必须留空**（保留逗号）。
3. **word (德语单词)**:
   - 保持德语拼写，名词首字母大写。
4. **cn (中文意思)**:
   - 对应的中文含义。
   - **注意**：中文里绝对**不能**出现英文逗号 `,`，如果需要分隔，请用中文顿号 `、` 或分号 `;` 代替。
5. **forms (特殊变化)**:
   书上标注了一些特殊变化 不过较为简略 请识别后加以理解，以如下形式填入
   - 名词：填写复数形式，格式统一为 `Pl. 复数词` (例如: Pl. Äpfel)。
   - 动词：填写第三人称单数变位，格式为 `er 变位词` (例如: er läuft)。
   - 如果书上没标，留空。
   - **注意**：绝对不能包含英文逗号。
6. **example (例句)**:
   - 如果图片中对应单词有例句，请录入。 如果是两个单词对应一个例句，那么这两个单词的这一栏都填写这个例句
   - **重要警告**：例句中如果包含逗号，请**必须**替换为分号 `;` 或者直接去除，否则会破坏 CSV 格式。

### 3. 输出示例（供参考）：
type,gender,word,cn,forms,example
n,der,Apfel,苹果,Pl. Äpfel,Der Apfel schmeckt gut.
n,die,Katze,猫,Pl. Katzen,Die Katze schläft.
v,,laufen,跑,er läuft,Er läuft sehr schnell.
adj,,schön,美丽的/好看的,,Das Wetter ist schön.
n,das,Buch,书,Pl. Bücher,Ich lese das Buch.

### 4. 任务开始：
请识别我上传的图片或表格，并输出符合上述规则的 CSV 内容。
```
